# D.I.N.O.


**Игра Chrome Dino, реализованная полностью на цифровой логике в Logisim**

---

## Описание

DINO — это воссоздание знаменитой офлайн-игры Chrome Dino из браузера Google Chrome, полностью реализованное на цифровых логических схемах в среде Logisim. Проект демонстрирует, как базовые логические элементы и цифровые платформы могут быть объединены для создания полноценной интерактивной игры с физикой, процедурной генерацией и системой рекордов на базе процессора CdM-8.

## Возможности

- **Полноценный геймплей**: прыжки, приседания, уклонение от препятствий
- **Несколько типов препятствий**: маленькие, большие и широкие кактусы, анимированные птицы
- **Физика прыжка**: реализация кинематического уравнения (y = y₀ + v₀t - ½gt²)
- **Смена дня и ночи**: переключение каждую половину минуты реального времени
- **Система рекордов**: пузырьковая сортировка на процессоре CdM-8
- **Подсчёт очков**: BCD-счётчики для очков, уровней и таймера (ММ:СС)
- **Динамическая сложность**: увеличение скорости с ростом уровня

## Требования

- Logisim 2.7.1 или новее
- Java Runtime Environment (JRE 8+)

## Быстрый старт

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/dino-logisim.git
cd dino-logisim
```

# Открыть основной файл проекта в Logisim
# File -> Open -> proj.circ

### Запуск игры

1 Откройте proj.circ в Logisim
2 Переключитесь в режим симуляции (инструмент «рука»)
3 Включите такты: Ctrl+K или моделирование -> такты включены
4 Установите необходимую частоту тактового генератора (512/1024 Гц)
5 Введите никнейм в клавиатуру таблицы результатов (до 4 символов)
6 В клавиатуре главного дисплея снимите паузу
7 Начинайте игру!


## Управление
### В игре
|------------------------------------------------------------|
| Клавиша                       | Действие                   |
|------------------------------------------------------------|
| Пробел / W / Стрелка вверх    | Прыжок                     |
| S / Стрелка вниз              | Приседание (переключатель) |
| P / Pause                     | Пауза                      |   
| R / Reset                     | Перезапуск                 |
|------------------------------------------------------------|

### Ввод никнейма
|------------------------------------------------------------|
| Клавиша                       | Действие
| -----------------------------------------------------------|
| A-Z, А-Я                      | Ввод символов (макс. 4)    |
| Enter                         | Подтверждение никнейма     |
| Clear                         | Очистка поля ввода         |
| -----------------------------------------------------------|
